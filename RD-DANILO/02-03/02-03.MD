# Modelagem

- Modelar é representar a realidade de forma estruturada e simplificada, traduzindo as necessidades do negocio para uma linguagem tecnica organizada.

Ou seja é o mesmo como criar uma "planta de uma casa" exemplo: se voce vai contruir uma casa(representação da realidade), voce precisa de uma planta(modelagem);

```bash
    É de suma importancia fazer a modelagem para evitar riscos e muitas vezes erros futeis
```
