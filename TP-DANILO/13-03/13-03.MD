# POLIMORFISMO

- POLI     -> Muitas
- MORFISMO -> Formas

<br>

animal -> Gato
animal -> Cachorro

    O código abaixo demonstra como usar `virtual` e `override` para permitir que classes filhas implementem comportamentos específicos.

```csharp
public class Animal
{
    // O modificador 'virtual' permite que o método seja sobrescrito
    public virtual void EmitirSom()
    {
        Console.WriteLine("Som genérico de animal!");
    }
}

public class Cachorro : Animal
{
    //Usa override é ultilizado nas classes filho para indicar que vai implementar/substituir no metodo de maneira diferente da classe pai
    public override void EmitirSom()
    {
        Console.WriteLine("Au Au!");
    }
}

public class Gato : Animal
{
    public override void EmitirSom()
    {
        Console.WriteLine("Miau Miau!");
    }
}

```

## POLIMORFISMO OVERLOADING(SOBRECARGA)

```csharp
public int Soma(int n1, int n2)
{
    return n1 + n2;
}

public double Soma(double n1, double n2)
{
    return n1 + n2;
}

```
