# Mapeamento Sistematico do MER ao Relacional

    Entidade       -> Tabela
    Atributo       -> Coluna
    Identificador  -> Chave Primaria(PK)
    Relacionamento -> Chave Estrangeira (FK) ou Nova Tabela

Todo relacionamento N para N vira uma nova tabela
- 

```mermaid

    erDiagram
    Usuario ||--o{ Post : cria
    Usuario ||--o{ Comentario : escreve
    Post ||--o{ Comentario : possui

    Usuario ||--o{ Mensagem : envia
    Usuario ||--o{ Mensagem : recebe

    Usuario ||--o{ Amizade : solicita
    Usuario ||--o{ Amizade : recebe

    Usuario }o--o{ Post : curte

    Usuario {
        int id
        string nome
        string email
        string senha
        date data_criacao
    }

    Post {
        int id
        int usuario_id
        string conteudo
        datetime data_postagem
    }

    Comentario {
        int id
        int usuario_id
        int post_id
        string texto
        datetime data_comentario
    }

    Curtida {
        int usuario_id
        int post_id
        datetime data_curtida
    }

    Amizade {
        int id
        int solicitante_id
        int destinatario_id
        string status
        datetime data_solicitacao
    }

    Mensagem {
        int id
        int remetente_id
        int destinatario_id
        string texto
        datetime data_envio
    }
```
