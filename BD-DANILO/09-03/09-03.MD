# Resolução

    ▭ -> Retângulo (Entidade)

    <> -> Losango   (Relacionamento)

    ⬭ -> Eclipse   (Atributos)

<br>

MER -> (Modelo Entidade de Relacionamento)

DER -> (Diagrama Entidade de Relacionamento)

<br>

Entidade

    CLIENTE: Id, Nome, Idade, Sexo, Altura e peso
    TREINADOR: Id, Nome, Especialidade
    EXERCÍCIO: Id, Nome, Descricao      OBD(PODE REQUERER O USO DE EQUIPAMENTOS)
    EQUIPAMENTO: Id, Nome, Descricao

<br>

    CLIENTE - TREINO    |     TREINADOR - TREINO
        1        N                1          N
