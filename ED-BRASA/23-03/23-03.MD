# Funções anônimas no Javascript

- Funções anônimas em Javascript são funções definidas <u>*sem um nome identificador,*</u> geralmente criadas no local de uso, atribuidas á variaveis ou passadas como argumentos

## Exemplo 1
```javascript
    let mostrarMensagem = function(){
        document.write("Olá")
    }
    mostrarMensagem()
```

<br>

## Exemplo 2
```javascript
    let funcao = function(){
        somar(1, 2)
    }
    function somar(p1, p2){
        let res = p1 + p2
        document.write("A soma é: " + res)
    }
    funcao()
```
