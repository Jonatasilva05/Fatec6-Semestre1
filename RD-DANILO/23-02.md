# Aula 2

## Conceito ACID no banco de dados

- Atomicidade; A transação é uma unidade unica: tudo ocorre ou nada ocorre. Ex Transferencia bancaria que falha e devolve o valor

- Consistência; O banco migra de um estado valido para outro estado valido. Ex impedir que o estoque de livros fiquem negativos

- Isolamento; Transação simultâneas não interferem entre si. Ex Dois usuarios comprando o ultimo ingresso disponivel

- Durabilidade; Alterações confirmadas são permanentes, mesmo apos falhas. Ex Cadastro salvo apos queda subita de energia.
