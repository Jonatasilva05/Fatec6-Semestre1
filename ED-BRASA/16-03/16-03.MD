# ALGORITMO DE ORDENAÇÃO BUBBLE SORT
O bubble sort é um algoritmo simples que ordena uma lista comparando elementos adjacentes e trocando-os se estiverem na ordem errada. O processo vai se repetinfo até que esteja ordenada, fazendo os valores "borbullarem" a cada passagem

## Exemplo ordenando array
```javascript
    const max = s
    let v = []
    let i, j, aux, valor

    //grava os elementos
    for(i=0; i<max; i++){
        valor = number(window.prompt("Digite o valor"))
        v.push(valor)
    }

    //lista o array original
    for(i=0; i<max; i++){
        document.write(i + " " + v[i] + "<br>")
    };

```
