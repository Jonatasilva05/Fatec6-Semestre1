## Exercicio
- 1 Elabore uma pagina que leia um numero e, em seguida, mostre se o numero é positivo ou negativo, sabendo que um numero é negativo quando o mesmo é menor que zero

- 2 Elabore uma pagina que leia um numero e verifica se ele é par um impar. Quando for par armazenar o valor dentro de uma variavel P e quando for impar, armazenar o valor dentro de uma variavel I. Exibir os valores de P e I

- 3 Faça uma pagina para entrar com o codigo e mostrar o cargo conforme tabela abaixo:
codigo - | - cargo
1      - | - secretaria
2      - | - gerente
3      - | - diretor

- 4 Faça uma pagina usando a repetição FOR para exibir a sequencia de 7 a 200

- 5 Faça uma pagina usando a repetição FOR para exibir a sequencia entre dois numeros queisquer digitados pelo usuario
