# INTERFACE NA POO(Programação Orientada a Objeto)

- A interface são contratos que definem um conjunto de metodos que ua classe deve implementar

IPagamento
(interface -> contrato)


<br>

EXEMPLO DE CRIAÇÃO DE INTERFACE
    
    public interface IAnimal
    {
        void FazerBarulho();
    }

---

    public class Cachorro : IAnimal
    {
        public void FazerBarulho()
        {
            Console.WriteLine("Cachorro faz Au!");
        }
    }

---
