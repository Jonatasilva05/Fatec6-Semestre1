# Banco de Dados 1

## Descrição

A disciplina de **Banco de Dados 1** fornece os fundamentos para o projeto, manipulação e gerenciamento de dados em sistemas relacionais. Através dessa matéria, o aluno aprenderá a modelar dados, utilizar SQL para interagir com bancos de dados e a entender os princípios de normalização, transações e integridade dos dados.

### Para que serve:
Essa matéria serve para capacitar os alunos a construir e gerenciar bancos de dados eficientes, essenciais para o funcionamento de sistemas modernos, como sistemas de gerenciamento empresarial, e-commerce, aplicativos móveis, entre outros.

### Onde será tratada e utilizada:
- **Empresas de TI**: Desenvolvimento de sistemas com alta demanda de manipulação de dados (CRM, ERP, sistemas financeiros).
- **Startups**: Desenvolvimento de soluções de dados para pequenos negócios que precisam de uma estrutura de banco de dados eficiente e escalável.
- **Consultoria de TI**: Proposta e implementação de soluções de banco de dados para clientes, otimizando consultas e operações de dados.

### Objetivos:
- Entender os conceitos de modelagem de dados e normalização.
- Aprender a criar e manipular bancos de dados usando SQL.
- Compreender a importância de transações e segurança em banco de dados.

### Tópicos:
1. **Modelagem de Dados**:
   - Entidade-Relacionamento (ER), diagramas ER.
   - Definição de tabelas e relacionamento entre elas.
2. **Normalização**:
   - Processo de normalização de dados até as formas normais (1NF, 2NF, 3NF).
   - Prevenção de redundância e anomalias.
3. **Linguagem SQL**:
   - Comandos básicos: SELECT, INSERT, UPDATE, DELETE.
   - Consultas complexas, joins, subconsultas.
4. **Transações e Controle de Concorrência**:
   - Propriedades ACID (Atomicidade, Consistência, Isolamento e Durabilidade).
   - Controle de transações e locks.
5. **Backup e Recuperação de Dados**:
   - Técnicas de backup.
   - Estratégias para recuperação de dados.
