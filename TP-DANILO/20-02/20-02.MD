# 4 PILARES DA PROGRAMAÇÃO ORIENTADA A OBJETO

- Encapsulamento;
- Herança;
- Polimorfismo;
- Abstração

## Encapsulamento:
- Proteje os dados internos de um Objeto, ocultando-se de interações externas diretas e fornecendo interfaces publicas (metodos)

## Herança:
- Permite que uma classe (subclasse/filha) herde atributos e metodos de outra classe (superclasse/pai), sendo assim permitindo reaproveitamento de código

## Polimorfismo:
- Capacidade de objetos de diferentes classes serem tratados de forma uniforme por meio de uma interface comum, permitindo que o mesmo método tenha comportamentos diferentes.

## Abstração:
- Foca nos aspectos essenciais de um objeto, escondendo detalhes complexos de implementação. Transforma conceitos do mundo real em modelos simplificados no código.
