# Vetores ou Arrays

---

### Arrays

- Arrays é um recurso que permite a criação de uma variável multivalorada, sedno ela a mais simples dentre as estruturas de dados. Muitas vezes, é formada por dados do mesmo tipo e os dados são acessados através de sua posição ou indice.

Quando o array é formado por valores iguais como na tabela abaixo o array é chamado de Homogêneo, Senão for iguais então sera chamado de Heterogêneo

---

### Vetores

| 00 | 01 | 02 | 03 | 04 | 05 |
| :--- | :---: | :---: | :--- | :---: | ---: |
| 55 | 73 | 36 | 47 | 56 | 58 |

---

### EXEMPLO DE SCRIPT

```javascript
<script>
    const max = 6
    let v = []
    let i, valor, encontrou

    // Grava no Vetor ou Arrays
    for(i = 0; i < max; i++){
        valor = Number(window.prompt("Digite Numero"))
        v.push(valor)
    }

    // Lista os elementos no vetor
    Document.write("br")

    for(i = 0; i < max; i++){
        Document.write(`${i}: ${v[i]}`+'<br>')
    }

    // Verifica se o valor 7 existe no vetor ou arrays
    Document.write("<br>")
    encontrou v = False
    for(i = 0; i < max; i++){
        if(v[i] == 7){
            encontrou = True
            Document.write(`O valor 7 foi encontrado em ${i}` + '<br>')
        }
    }
    if(!encontrou){
        Document.write("O valor 7 não foi encontrado")
    }
</script>
